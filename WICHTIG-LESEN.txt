❓ PROBLEME?

▸ "JWT validation failed"

  → Neu einloggen
  → Passiert nur EINMAL beim ersten Start

▸ "Chat antwortet nicht"
  → API Keys in backend\.env hinzufügen

▸ "Port bereits verwendet"
  → Andere Instanzen schließen
  → Task Manager → Python/Node beenden

▸ Alles andere
  → Lesen Sie: README-DEUTSCH.md
